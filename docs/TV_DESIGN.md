## 一、概述

### 1.1 定位

TV 端是 LinPlayer 的独立平台版本，覆盖：
- **Android TV**（智能电视、电视盒子、投影仪）
- **Apple TV**（tvOS）
- **平板**（Android 平板 / iPad）

平板使用 TV 端 UI，额外支持触摸回退操作。

### 1.2 核心原则

- **遥控器驱动**：所有交互以 D-pad + 确认键为默认输入方式
- **焦点放大**：远距离观看需要物理尺寸变化的视觉反馈
- **单交互色**：仅使用品牌色 `#5B8DEF` 作为焦点/交互指示（参考 Apple 单 Action Blue 原则）
- **UI 后退**：封面/视频优先，UI 元素极简隐入背景
- **无阴影**：所有 UI 元素无阴影，焦点用 outline + 光晕（参考 Apple 无阴影原则）
- **强制深色**：TV 端始终深色模式，无视系统设置

### 1.3 目录结构

## 二、架构决策

### 2.1 平台范围

| 平台 | 优先级 | 输入方式 | 构建产物 |
|------|--------|----------|----------|
| Android TV | P0 | 遥控器 + 平板触摸 | `linplayer-tv.apk` |
| Apple TV | P1 | 遥控器 | `linplayer-tv.ipa` |
| Android 平板 | P0 | 触摸回退 | 复用 TV APK |

### 2.2 代码复用策略

- **复用 `core/`**：API 层、播放器服务、共享 Provider 完全复用
- **新增 `tv/`**：TV 专用 UI、专用 Provider（焦点管理、导航状态）
- **拆分平台特定 Provider**：
  - 手机端：手势状态、PiP 状态、触摸相关
  - TV 端：焦点状态、遥控器输入、TV 导航状态
  - 共享：服务器列表、播放进度、观看记录、媒体库数据

### 2.3 技术栈

| 技术 | 用途 |
|------|------|
| Flutter Focus | 焦点导航基础设施 |
| FocusTraversalPolicy | 自定义方向导航 |
| media_kit | MPV 播放器（Android TV + Apple TV） |
| ExoPlayer | Android TV 备选播放器 |
| Riverpod | 状态管理（复用） |
| GoRouter | TV 端路由 |

## 五、焦点系统

### 5.1 核心策略

基于 Flutter 原生 `Focus` + 自定义 `FocusTraversalPolicy`，不另起焦点管理器。

### 5.2 关键组件

#### `TVFocusable`（通用焦点包裹）

```dart
class TVFocusable extends StatelessWidget {
  final Widget child;
  final VoidCallback? onSelect;
  final bool autoFocus;
  
  // 焦点态：放大 1.15x + 白色边框 + 品牌色光晕
  // 非焦点态：透明度 0.8
  // 动画：250ms，Curves.easeInOut
}
```

#### `TVFocusRow`（横向焦点行）

```dart
class TVFocusRow extends StatelessWidget {
  final List<Widget> children;
  final double spacing;
  
  // 行内焦点移动：左右键横向移动
  // 行内滚动：焦点项自动滚动到可见区域
  // 跨行对齐：按上下键时，新行自动对齐到最接近的列
}
```

#### `TVFocusGrid`（焦点网格）

```dart
class TVFocusGrid extends StatelessWidget {
  final int crossAxisCount;
  final List<Widget> children;
  
  // 网格内焦点移动：上下左右键
  // 到边缘自动加载下一页
}
```

### 5.3 焦点记忆

- 导航栏 ↔ 内容区：离开内容区时保存当前焦点索引，返回时恢复
- 页面切换：返回上一页时，恢复离开时的焦点位置
- 实现：使用 `FocusNode` 的 `requestFocus()` + 手动索引栈

### 5.4 焦点导航规则

| 场景 | 行为 |
|------|------|
| 首页内容区按"上" | 焦点移到 Hero Banner 的"播放"按钮 |
| 首页内容区按"左" | 焦点移到导航栏当前项 |
| 导航栏按"右" | 焦点返回内容区（恢复离开时的位置） |
| 播放页控制层按"上" | 焦点从底部控制栏移到进度条 |
| 播放页控制层按"下" | 焦点从进度条移到底部控制栏 |
| 列表按"上"到顶部 | 焦点移到筛选栏/导航栏 |
| 列表按"下"到底部 | 触发加载下一页（或显示"已加载全部"） |

### 6.3 Hero 内容策略

- **优先**：继续观看（第一个条目，如果正在观看或有新集）
- **降级**：无继续观看时显示随机推荐（8 个，固定从第一个开始）
- **下方行**：继续观看（剩余条目，不含 Hero 已展示的第一个）/ 媒体库 / 各媒体库最近更新
- **新集**：有更新时排到第一个展示
- **已看完当前集**：如果已看完当前集，自动将下一集（未看）加入"继续观看"
- **轮播**：
  - 自动轮播：10 秒间隔，淡入淡出 500ms，`Curves.easeInOut`
  - 手动切换：遥控器左右键，焦点在 Hero 时切换
  - 暂停轮播：焦点离开 Hero 区域时暂停自动轮播
  - 触摸模式：左右滑动切换（平板）
  - 指示器：底部圆点，当前项品牌色实心，非当前项 `#FFFFFF40`

### 6.4 继续观看行策略

- **包含内容**：
  - 有进度未看完（进度 1%-99%）
  - 已看完当前集，下一集未看（自动加入继续观看）
- **不包含**：
  - 整季看完且无新集（暂时隐藏）
- **排序**：
  - 有新集更新 → 排到第一个
  - 有进度未看完 → 按最近观看时间排序
  - 已看完当前集，下一集未看 → 按看完时间排序
- **卡片展示**：
  - 封面 + 标题 + 进度条（底部品牌色）+ 集数标签
  - 新集：右上角"新"标签（红色/品牌色）
  - 已看完但下一集未看：显示"S1E6 · 下一集"标签

## 七、播放页设计

### 7.1 双模式架构

| 模式 | 触发 | 视觉 |
|------|------|------|
| 遥控器模式 | 检测到遥控器按键 | 显示焦点边框 + 焦点放大 |
| 触摸模式 | 检测到触摸输入 | 隐藏焦点边框，直接点击触发 |

### 7.2 控制层显隐

- 遥控器按键 / 触摸屏幕 → 控制层淡入（300ms）
- 5 秒无操作 / 触摸空白区域 / 按返回键（控制层已显示时）→ 控制层淡出（400ms）

### 7.3 控制层布局

```
┌─────────────────────────────────────────────┐
│ [←] 标题(滚动)  [跳过] [···]             │  ← 顶部栏（透明）
│                                             │
│                                             │
│                                             │
│                                             │
│          [进度条 + 时间]                    │  ← 进度条区域
│                                             │
│ [上集] [快退] [▶/⏸] [快进] [下集] [字] [音] [更]│  ← 底部控制栏
└─────────────────────────────────────────────┘
```

### 7.4 顶部栏

| 位置 | 元素 | 说明 |
|------|------|------|
| 左侧 | 返回按钮 | 退出播放页 |
| 中间 | 集标题 | 太长则滚动显示 |
| 右侧 | 跳过按钮 | 片头/片尾跳过设置（如果有章节数据） |
| 右侧 | 更多按钮 | 呼出右侧滑入面板（字幕/音轨/倍速/画面比例/内核切换/章节等） |

### 7.5 进度条区域

- 距底部：120dp（给底部控制栏留空间）
- 左右内边距：48dp
- 进度条：高度 4dp（焦点时 8dp），已播放部分品牌色
- 时间：左侧当前时间，右侧剩余时间（点击切换）
- 遥控器焦点在进度条时：左右键快退/快进（固定步进）
- 触摸：直接拖拽进度条

### 7.6 底部控制栏

| 位置 | 按钮 | 功能 |
|------|------|------|
| 左侧 | 上一集 | 切换集 |
| 左侧 | 快退 | 10s（默认） |
| 中央 | 播放/暂停 | 播放/暂停 |
| 右侧 | 快进 | 10s（默认） |
| 右侧 | 下一集 | 切换集 |
| 右侧 | 字幕 | 呼出字幕设置面板 |
| 右侧 | 音轨 | 呼出音轨设置面板 |
| 右侧 | 更多 | 呼出更多面板 |

### 7.7 快进/快退策略

| 输入 | 步进 | 说明 |
|------|------|------|
| 短按左/右 | 10s | 默认步进，可在设置调整 |
| 长按左/右 | 加速连续步进 | 速度逐渐加速（10s → 30s → 1min → 5min） |
| 进度条焦点 + 左/右 | 10s | 精确步进 |

### 7.8 触摸手势（平板模式）

保留手机端全部手势：

| 手势 | 功能 |
|------|------|
| 双击左侧 | 快退（可在设置自定义） |
| 双击中间 | 播放/暂停（可在设置自定义） |
| 双击右侧 | 快进（可在设置自定义） |
| 长按屏幕 | 临时快进（倍速可在设置自定义） |
| 左侧上下滑 | 调节亮度 |
| 右侧上下滑 | 调节音量 |
| 左右滑 | 调节进度 |

#### 播放页控制层焦点导航

- 默认焦点：播放/暂停按钮（底部控制栏中央）
- 按"上"键：焦点从底部控制栏 → 进度条
- 按"下"键（焦点在进度条）：回到底部控制栏
- 按"左/右"键（焦点在进度条）：快退/快进（固定步进）
- 长按左右键（焦点在进度条）：连续步进（加速）
- 按"上"键（焦点在进度条）：焦点到顶部栏（返回/跳过/更多）
- 按"下"键（焦点在顶部栏）：回到进度条
- 按"返回"键（控制层已显示）：隐藏控制层
- 触摸空白区域：隐藏控制层
- 无操作 5 秒：自动隐藏控制层（400ms 淡出）
- 长按进度条：直接拖拽到目标位置（触摸模式）或连续步进（遥控器模式）

### 7.9 更多面板（右侧滑入）

| 分组 | 项 |
|------|-----|
| 播放设置 | 倍速、画面比例、内核切换 |
| 轨道设置 | 字幕、音轨、次字幕 |
| 其他 | 投屏、统计信息、定时关闭、截图、锁定 |

### 7.10 暂停状态

- 暂停时：视频画面定格，控制层显示
- 屏幕中央显示半透明圆形背景 + 白色暂停图标
  - 圆形背景：直径 80dp，品牌色 `#5B8DEF` 40% 透明度
  - 暂停图标：白色，40dp
  - 动画：淡入 200ms
  - 恢复播放：图标淡出 200ms
- 控制层中的播放按钮显示为"▶"（三角形）
- 触摸空白区域或按确认键：恢复播放

### 7.11 章节跳转面板

- 章节入口在"更多"面板中（如果有章节数据）
- 章节面板：右侧滑入，列出所有章节
  - 每项：章节名 + 时间戳
  - 焦点选中跳转
- 进度条显示章节标记：小竖线，品牌色，悬浮提示章节名
- 无章节数据时，章节入口隐藏
- 跳过片头/片尾：章节面板顶部显示"片头"和"片尾"快速跳转（如果有）

### 7.12 播放完毕处理

- 播放完毕（进度 100%）→ 自动上报 Emby 播放停止
- 自动播放下一集：默认开启（设置可关闭）
- 自动播放时：淡入下一集（300ms），不显示播放页控制层
- 无下一集时：停在结尾画面，显示控制层（方便用户操作）
- 播放页控制层显示"播放完毕"提示（3 秒，Toast）
- 触摸模式：点击"下一集"按钮或滑动选择其他集

## 二十一、日志系统

### 21.1 日志存储

- TV 端日志存储在本地文件（`path_provider` 的文档目录）
- 日志文件按日期滚动，保留最近 7 天
- 日志格式：时间戳 + 级别 + 模块 + 消息
- 日志级别：DEBUG / INFO / WARNING / ERROR

### 21.2 日志下载

TV 端无法方便查看日志，通过局域网 HTTP 服务器 + 手机扫码下载。

#### 方案：内置临时 HTTP 服务器

- **设置页 → 关于 → 日志** 显示二维码
- 二维码内容：局域网下载 URL（如 `http://192.168.1.100:8080/logs`）
- TV 端启动临时 HTTP 服务器（只在日志页面存在时运行）
- 手机扫描二维码 → 浏览器打开 → 下载日志文件

#### 服务器参数

| 参数 | 值 |
|------|-----|
| 端口 | 8080（如果占用则随机） |
| 绑定 IP | 仅局域网 IP（`192.168.x.x` / `10.x.x.x`），不绑定公网 |
| 路由 `/logs` | 下载全部日志（zip 压缩） |
| 路由 `/logs/latest` | 下载最近 1 天日志 |
| 路由 `/logs/session` | 下载当前会话日志 |

#### 页面设计

```
┌─────────────────────────────────────────────┐
│  导航栏 │ 日志下载                           │
│         │                                  │
│         │   ┌────────────────────────┐     │
│         │   │                        │     │
│         │   │     [二维码大图]        │     │
│         │   │                        │     │
│         │   └────────────────────────┘     │
│         │                                  │
│         │   http://192.168.1.100:8080/logs │
│         │                                  │
│         │   [ 下载全部日志 ]              │
│         │   [ 下载最近1天 ]               │
│         │   [ 下载当前会话 ]               │
│         │                                  │
│         │   服务器运行中...               │
│         └──────────────────────────────────┘
```

| 参数 | 值 |
|------|-----|
| 二维码 | 中央大图，200x200dp |
| URL 文字 | 16sp，可复制（平板触摸） |
| 下载按钮 | 纵向排列，56dp 高，品牌色背景 |
| 状态提示 | 14sp，灰色 |
| 安全提示 | "仅局域网可用，退出页面后服务器关闭" |

#### 生命周期

- 进入日志页面 → 启动 HTTP 服务器
- 退出日志页面（返回键/导航切换） → 关闭 HTTP 服务器
- 应用后台 → 保持服务器运行（如果页面仍在）
- 应用退出 → 关闭服务器

#### 日志保留策略

| 策略 | 说明 |
|------|------|
| 保留周期 | 7 天滚动 |
| 单个文件大小 | 最大 5MB，超过则新建文件 |
| 总大小限制 | 50MB，超过则删除最旧文件 |
| 日志格式 | `[2024-06-13 10:30:45] [INFO] [Player] 开始播放` |
